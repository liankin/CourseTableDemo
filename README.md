# CourseTableDemo
简易课程表
==

博客地址：
--
[CSDN博客]http://blog.csdn.net/an_nal/article/details/78923527

实现的功能：
--
1.固定课节数为 12 节； 
2.新增、修改课节时间； 
3.新增课程，单击已有的课程进行修改课程信息，长按进行删除； 
4.课程信息和课节时间的数据存储在SQLite数据库中；
